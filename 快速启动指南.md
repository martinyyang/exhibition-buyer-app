# 🚀 快速启动指南

## 立即运行App

### 前提条件检查

1. **确认Flutter已安装**
   - 打开**新的**命令行窗口（让环境变量生效）
   - 运行：
     ```bash
     flutter --version
     ```
   - 应该看到：`Flutter 3.24.5`

2. **准备Android设备**
   - **选项A**：连接Android手机（开启USB调试）
   - **选项B**：启动Android模拟器

## 🔧 常见问题

### Q1: 提示 "No connected devices"
**原因**: 没有连接Android设备

**解决**:
1. 确保USB调试已开启
2. 或者启动Android模拟器
3. 运行 `flutter devices` 查看设备列表

### Q2: 第一次运行很慢
**原因**: Flutter需要下载Gradle依赖

**解决**: 耐心等待，可能需要5-10分钟（只有第一次）

### Q3: 提示 "Supabase connection failed"
**原因**: 网络问题或配置错误

**解决**:
1. 检查网络连接
2. 确认 `.env` 文件配置正确
3. 打开 `.env` 文件，确认：
   - `SUPABASE_URL=https://ppwjblvnixqeympfcqgs.supabase.co`
   - `SUPABASE_ANON_KEY=eyJhbGci...` (完整的密钥)

### Q4: 找不到 flutter 命令
**原因**: 环境变量未生效

**解决**: 
1. 关闭当前命令行窗口
2. 打开**新的**命令行窗口
3. 重试

### Q5: 想要热重载（修改代码实时更新）
**操作**:
- 按 `r` - 热重载
- 按 `R` - 完全重启
- 按 `q` - 退出

## 🎮 App使用说明

### 首次使用流程

1. **注册账号**
   - 打开App
   - 点击"注册"
   - 输入邮箱、密码
   - 选择角色：买手 或 领班

2. **创建场次**（需要领班权限）
   - 点击"新建场次"
   - 输入展会名称、地点、时间

3. **添加摊位**
   - 进入某个场次
   - 点击"新建摊位"
   - 输入摊位号

4. **拍照记录**
   - 进入摊位详情
   - 点击"拍照"
   - 拍摄供应商商品
   - 自动上传到云端

5. **标记旗子**
   - 在照片上点击
   - 添加旗子标记
   - 记录商品信息和价格

## 💾 数据说明

### 数据存储位置
- **照片**: Supabase Storage（云端）
- **数据库**: Supabase PostgreSQL（云端）
- **本地缓存**: App内部存储（离线支持）

### 数据权限
- **买手**: 只能看到自己小组的数据
- **领班**: 可以看到所有小组的数据
- **管理员**: 完全访问权限
